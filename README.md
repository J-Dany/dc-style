## Requirements

You need [Node.js](https://nodejs.org/en) in order to modify and compile.

## Development

First, install the dependencies:

```bash
yarn install
```

## Build

```bash
yarn run compile
yarn run clean
# rm style.css.map                      // If you want to remove the file .map
```
